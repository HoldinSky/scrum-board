What is this project is about?

This web-service was created as a pet-project for sharpening programming skills and familiarization with opportunities of Spring Framework.

However the goal of this application is to simplify and improve future workflow. Inspiration was found in the Jeff Sutherland's book called SCRUM in which author describes the methodology for progressive and constantly improving workflow. As far as You could already understand SCRUM board is one of the tools of methodology.

This link leads to the GitHub repository of the project. Moreover, if You have any suggestions on improving user experience You may contact me via any convenient way offered at the top right of navigation bar.
